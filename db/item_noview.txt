// 裝備外觀隱藏數據庫
// 在這裡可以設置某些裝備, 在特定的時機下不顯示他們的外觀.
//
// 設計背景:
// 龍王蠻蠻所製作的部分圖檔在穿戴後雖然角色外觀看起來正常, 
// 但是只要打開裝備面板去查看的話, 可能就會導致客戶端直接崩潰.
// 為了避免客戶端崩潰, 所以添加此選項.
//
// 此數據庫可以使用 @reloaditemdb 重新讀取
// 但是若修改的是標記 1 或 2, 那麼玩家需要小退之後重新進入遊戲才能看到效果.
// 若修改的是標記 4 或 8, 則能夠直接生效, 無需小退.
//
// 數據庫格式說明:
// 物品編號,標記
//
// 以下為'標記'字段的可選值 (掩碼選項):
//   1 - 當自己查看自己裝備時, 若此道具為普通裝備, 則不在裝備面板的「普通裝備」選項卡中顯示外觀
//   2 - 當自己查看自己裝備時, 若此道具為時裝裝備, 則不在裝備面板的「時裝裝備」選項卡中顯示外觀
//   4 - 當別人查看自己裝備時, 若此道具為普通裝備, 則不在裝備面板的「普通裝備」選項卡中顯示外觀
//   8 - 當別人查看自己裝備時, 若此道具為時裝裝備, 則不在裝備面板的「時裝裝備」選項卡中顯示外觀
//
// 例如:
// 2220,1	// 穿戴漁夫帽時, 當自己查看自己裝備的時候, 不在裝備面板的「普通裝備」選項卡中顯示外觀
// 2220,4	// 穿戴漁夫帽時, 當別人查看自己裝備的時候, 不在裝備面板的「普通裝備」選項卡中顯示外觀
// 2220,5	// 穿戴漁夫帽時, 無論自己看還是別人看, 都不在裝備面板的「普通裝備」選項卡中顯示外觀
